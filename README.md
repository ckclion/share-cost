# ShareCost
## Description:

### DISCLAIMER:
Use in desktop/laptop for best experience!

### WHAT IS IT:
ShareCost is a webapp for users who need to share cost with their family or friends! Ever had a situation where everyone paid for their own food but its in a single bill? Using ShareCost, users can now easily see what is the final amount everyone has to either pay or receive from another person, saving time having to individually add and divide the cost for everyone.
